## 1. Introduction to Artificial Intelligence (AI)

Artificial Intelligence (AI) is a branch of computer science that enables machines to perform tasks that normally require human intelligence. These tasks include learning, reasoning, problem-solving, and decision-making.

## 2. Introduction to Machine Learning (ML)

Machine Learning (ML) is a subset of AI that allows computers to learn from data without being explicitly programmed. It improves its performance by identifying patterns and making predictions from data.

## 3. Data Exploration

Data Exploration is the process of understanding a dataset by analyzing its structure, features, and patterns. It helps identify missing values, outliers, and relationships before building a machine learning model.

## 4. Real-life Example of Data Exploration

In a student stress prediction dataset, data exploration helps analyze factors like study hours, sleep hours, and exam pressure. This helps understand how these factors affect students' stress levels.

## 5. Data Types

Data types define the kind of values stored in a dataset. Common data types include numerical (integers and decimals), categorical (labels), text, and date/time data.

## 6. Data Modality

Data modality refers to the different forms in which data is available. Examples include text, images, audio, video, and tabular data, each requiring different processing techniques.
